%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
QS space figure
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
Both folders runZn4k2resplus and runZn4k2resExtplus include the files to run and the data obtained for the space of QS configurations in Fig. 1 of the space of configurations paper. Each folder is subdivided into folders that correspond to different sectors of the space, done to parallelise the construction of the space. The code was run in the PPPL cluster.


%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
New QH configuration
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
The folder includes the VMEC input for the new QH configuration in the paper, as well as the equilibrium calculation, and the BOOZXFORM and NEO calculations with the Boozer content of |B| and the epsilon_eff for neoclassical transport.
